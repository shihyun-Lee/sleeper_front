# Sleeper_frontend

Sleerper 앱의 프론트 레포지토리입니다.
---

## 기술 스택

- kotlin

---


## 커밋 컨벤션

| 메시지      | 설명 |
|----------|------|
| feat     | 기능 추가 작업에 대한 커밋 |
| update   | 기능 수정 작업에 대한 커밋 |
| fix      | 버그 코드 수정에 대한 커밋 |
| refactor | 코드 리팩토링 작업에 대한 커밋|
| docs     | 문서 작업에 대한 커밋 |
